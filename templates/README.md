# Claude Code Templates

Ready-to-use templates for creating Claude Code extensions. Use these as starting points when building skills, subagents, or hooks.

## Templates

| Template | Description |
| --- | --- |
| [SKILL-TEMPLATE.md](SKILL-TEMPLATE.md) | Skill templates - reference, task, research, codegen, and script-powered patterns |
| [SUBAGENT-TEMPLATE.md](SUBAGENT-TEMPLATE.md) | Subagent templates - reviewer, debugger, domain expert, with-hooks, memory, background, and worktree patterns |
| [HOOKS-TEMPLATE.md](HOOKS-TEMPLATE.md) | Hooks templates - formatter, protector, notifier, auditor, verifier, and all hook event types |

## Usage

Copy the relevant template into your project's `.claude/` directory and customize it for your needs. See the full docs in [`claude-code-docs/`](../claude-code-docs/) for detailed guides:

- [skills.md](../claude-code-docs/skills.md) - Creating and using skills
- [sub-agents.md](../claude-code-docs/sub-agents.md) - Creating custom subagents
- [hooks.md](../claude-code-docs/hooks.md) - Hooks technical reference
