# Refactoring Commands

Code cleanup, refactoring, technical debt analysis, dependency auditing, context management, and linting commands.

Sources: [wshobson/agents](https://github.com/wshobson/agents) (MIT), [davila7/claude-code-templates](https://github.com/davila7/claude-code-templates) (MIT)

| File | Description |
|------|-------------|
| `refactor-clean.md` | Code refactoring with SOLID principles, clean code patterns, and before/after metrics |
| `tech-debt.md` | Technical debt inventory, impact assessment, and prioritized remediation plans |
| `deps-audit.md` | Dependency vulnerability scanning, license compliance, and supply chain security |
| `context-restore.md` | Semantic context restoration and memory rehydration for multi-agent workflows |
| `context-save.md` | Intelligent context capture and preservation across AI sessions |
| `cleanup-cache.md` | System cache cleanup (npm, Homebrew, Yarn, Docker) to free disk space |
| `lint.md` | Python linting and formatting with ruff, black, and other tools |
