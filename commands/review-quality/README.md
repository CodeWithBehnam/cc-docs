# Review and Quality Commands

Code review, PR enhancement, multi-agent review, accessibility, and quality analysis commands.

Source: [wshobson/agents](https://github.com/wshobson/agents) (MIT)

| File | Description |
|------|-------------|
| `full-review.md` | Multi-dimensional code review orchestrator covering architecture, security, performance, testing, and best practices |
| `pr-enhance.md` | PR optimization with comprehensive descriptions, review automation, and quality gates |
| `ai-review.md` | AI-powered code review combining static analysis, pattern recognition, and modern DevOps practices |
| `multi-agent-review.md` | Multi-agent code review orchestration with specialized domain expertise agents |
| `design-review.md` | Review existing UI code for design issues, usability problems, and improvements |
| `accessibility-audit.md` | WCAG accessibility audit with automated testing, color contrast, keyboard navigation, and screen reader checks |
