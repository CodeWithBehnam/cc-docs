# Team Management Commands

Agent team orchestration, standup notes, issue management, delegation, and business analysis commands.

Source: [wshobson/agents](https://github.com/wshobson/agents) (MIT)

### Agent Teams

Commands for multi-agent team orchestration. Requires `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1`.

| File | Description |
|------|-------------|
| `team-spawn.md` | Spawn agent teams using presets (review, debug, feature, security) or custom composition |
| `team-status.md` | Display team members, task status, and progress for an active team |
| `team-delegate.md` | Task delegation dashboard for managing team workload and rebalancing |
| `team-feature.md` | Parallel feature development with multiple agents and file ownership boundaries |
| `team-review.md` | Multi-reviewer parallel code review with specialized review dimensions |
| `team-debug.md` | Debug issues using competing hypotheses with parallel investigation |
| `team-shutdown.md` | Gracefully shut down an agent team and clean up resources |

### Collaboration and Planning

| File | Description |
|------|-------------|
| `issue.md` | GitHub issue resolution with systematic investigation, TDD implementation, and PR creation |
| `standup-notes.md` | Async standup notes generated from git commits, Jira tickets, and calendar events |
| `business-case.md` | Investor-ready business case document with market, solution, and financials |
| `financial-projections.md` | 3-5 year financial model with revenue, costs, cash flow, and scenario modeling |
| `market-opportunity.md` | Market opportunity analysis with TAM/SAM/SOM calculations |
