# Testing Commands

TDD cycles, test generation, API mocking, and performance testing commands.

Sources: [wshobson/agents](https://github.com/wshobson/agents) (MIT), [davila7/claude-code-templates](https://github.com/davila7/claude-code-templates) (MIT)

| File | Description |
|------|-------------|
| `tdd-cycle.md` | Full TDD workflow orchestrator with strict red-green-refactor discipline |
| `tdd-red.md` | Write comprehensive failing tests following TDD red phase principles |
| `tdd-green.md` | Implement minimal code to make failing tests pass in TDD green phase |
| `tdd-refactor.md` | Refactor code with confidence using comprehensive test safety net |
| `test-generate.md` | Automated unit test generation across multiple languages and frameworks |
| `api-mock.md` | API mocking framework for development, testing, and demonstration |
| `performance-optimization.md` | End-to-end application performance optimization from profiling to monitoring |
| `test.md` | Python test runner with pytest, unittest, and other testing frameworks |
