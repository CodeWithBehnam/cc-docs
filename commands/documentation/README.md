# Documentation Commands

Doc generation, code explanation, architecture documentation, and content creation commands.

Sources: [wshobson/agents](https://github.com/wshobson/agents) (MIT), [davila7/claude-code-templates](https://github.com/davila7/claude-code-templates) (MIT)

| File | Description |
|------|-------------|
| `doc-generate.md` | Automated documentation generation - API docs, architecture diagrams, user guides |
| `code-explain.md` | Code explanation with visual diagrams, step-by-step breakdown, and learning paths |
| `c4-architecture.md` | Generate C4 architecture documentation (Context, Container, Component, Code) from codebase |
| `create-blog-article.md` | Create SEO-optimized blog articles with AI-generated cover images |
