# Security Commands

Security scanning, compliance checking, SAST, XSS detection, and hardening commands.

Source: [wshobson/agents](https://github.com/wshobson/agents) (MIT)

| File | Description |
|------|-------------|
| `security-sast.md` | Static Application Security Testing for code vulnerability analysis across multiple languages |
| `security-hardening.md` | Comprehensive security hardening with defense-in-depth strategy across all layers |
| `security-dependencies.md` | Dependency vulnerability scanning, SBOM generation, and supply chain security |
| `compliance-check.md` | Regulatory compliance audit for GDPR, HIPAA, SOC2, PCI-DSS, and other standards |
| `xss-scan.md` | XSS vulnerability scanner for React, Vue, Angular, and vanilla JavaScript code |
