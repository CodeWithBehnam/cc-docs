# Development Commands

Feature development, scaffolding, component creation, and project setup commands.

Source: [wshobson/agents](https://github.com/wshobson/agents) (MIT)

| File | Description |
|------|-------------|
| `feature-development.md` | End-to-end feature development orchestrator with requirements, architecture, implementation, and testing phases |
| `full-stack-feature.md` | Full-stack feature development across backend, frontend, database, and infrastructure layers |
| `component-scaffold.md` | React/React Native component scaffolding with TypeScript, tests, styles, and accessibility |
| `create-component.md` | Guided UI component creation following established patterns and best practices |
| `python-scaffold.md` | Python project scaffolding with modern tooling (uv, FastAPI, Django) |
| `typescript-scaffold.md` | TypeScript project scaffolding with pnpm, Vite, Next.js, and modern tooling |
| `rust-project.md` | Rust project scaffolding with cargo tooling, module organization, and testing |
| `multi-platform.md` | Cross-platform feature development across web, mobile, and desktop with API-first architecture |
| `design-system-setup.md` | Initialize a design system with design tokens, component patterns, and documentation |
| `data-driven-feature.md` | Build features guided by data insights, A/B testing, and continuous measurement |
